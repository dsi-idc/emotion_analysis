Required packages:

1. commentjson

For the arabert based model, these are the required packages:
1. pyarabic
2. farasapy
3. transformers (latest version: >3.01)
4. pytorch 1.4.0 (I have used the following command in windows: conda install -n arabic_nlp pytorch==1.4.0 torchvision==0.5.0 cpuonly -c pytorch)
5. arabert (through git clone https://github.com/aub-mind/arabert to the project's folder)
