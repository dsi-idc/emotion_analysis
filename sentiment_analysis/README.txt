Required packages (recommend to install them in the listed order):
1. pytorch 1.4.0 (I have used the following command in windows: conda install -n arabic_nlp pytorch==1.4.0 torchvision==0.5.0 cpuonly -c pytorch)
2. transformers (latest version: >3.01)
3. farasapy (most have java installed)
4. pyarabic
5. arabert (through git clone https://github.com/aub-mind/arabert to the project's folder)
6. commentjson
7. emoji
